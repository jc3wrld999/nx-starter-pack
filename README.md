# NxStarterPack

- nx
- react
- nest
- postgresql

## Install


```bash
$> npm i -g @nrwl/cli@latest 
$> npm i -g yarn@latest
```
## Start the app

```bash
npx create-nx-workspace
```

To start the development server run `nx serve nx-starter-pack`. Open your browser and navigate to http://localhost:4200/. Happy coding!
